Python

Python

Python es un lenguaje popular de programación de alto nivel, orientado a objetos, usado para el desarrollo web y de aplicaciones informáticas, ciencia de datos, aprendizaje automático, análisis y automatización de datos e Inteligencia artificial. soporta el uso de módulos y paquetes para un código puede ser reutilizado en varios proyectos. es un lenguaje de programación flexible con librerías, las cuales son conjunto de implementaciones funcionales que ofrecen una interfaz bien definida para la funcionalidad que se invoca.

Java

Java

Java se utiliza para desarrollar aplicaciones móviles, aplicaciones web, aplicaciones de escritorio, juegos y mucho más. Java es un lenguaje que funciona en diferentes plataformas (Windows, Mac, Linux, Raspberry Pi, etc.), seguro, rápido y potente. Tiene un gran apoyo de la comunidad (decenas de millones de desarrolladores). Java es un lenguaje orientado a objetos que proporciona una estructura clara a los programas y permite reutilizar el código, lo que reduce los costos y tiempos de desarrollo.

VB.NET

VB.NET

Visual Basic .NET (también llamado VB.NET) es un lenguaje de programación orientado a objetos desarrollado por Microsoft. Su uso agiliza y simplifica la creación de aplicaciones .NET, este lenguaje permite crear aplicaciones de escritorio, Web y móviles. Brinda un completo número de características para hacer que el desarrollo de aplicaciones sea realmente rápido. proporciona el entorno de ejecución Common Language Runtime (CLR), el cual hace posible aplicaciones sólidas, estables , seguras y mas rapidas ya que libera recursos y se encarga de administrar la memoria.

PHP

PHP

PHP es un lenguaje de secuencias de comandos de servidor y una poderosa herramienta para crear páginas web dinámicas e interactivas. está centrado en la programación de scripts del lado del servidor. Puede generar contenido de página dinámico, puede recopilar datos de formulario, puede administrar datos en su base de datos Se ejecuta en varias plataformas (Windows, Linux, Unix, Mac OS X, etc.), es compatible con casi todos los servidores que se utilizan hoy en día y soporta una amplia gama de bases de datos. WordPress, el sistema de administración de contenido más popular está hecho con el lenguaje de programación PHP

MySQL

MySQL

MySQL es un sistema de gestión de bases de datos relacional lo que le permite tener mayor velocidad y flexibilidad, es de las más populares para entornos de desarrollo web como WordPress, su popularidad como aplicación web está muy ligada a PHP, Grandes sitios web como Facebook, Twitter, etc. utilizan MySQL, las bases de datos en MySQL son muy rápidas, Utilizan varias capas de seguridad, necesitan pocos recursos de CPU o RAM y son compatibles con Linux y Windows.

JavaScript

JavaScript

JavaScript es un lenguaje de programación web muy popular, se utiliza principalmente del lado del cliente, implementado como parte de un navegador web permitiendo mejoras en la interfaz de usuario y páginas web dinámica. Es uno de los 3 lenguajes fundamentales para el desarrollo web junto a HTML y CSS, JavaScript se utiliza para programar el comportamiento de las páginas web, todos los navegadores modernos interpretan el código JavaScript integrado en las páginas web. junto con ayuda de otras tecnologías como AJAX puede utilizarse para enviar y recibir información del servidor

HTML

HTML

HTML es el lenguaje de marcado estándar para crear sitios web, es el componente más básico de la Web, su proposito es describir la estructura de una página Web a travez de una serie de elementos. La funcion de un navegador web es leer documentos HTML y mostrarlos correctamente.Este lenguaje nos ofrece una gran adaptabilidad, una estructuración lógica y es fácil de interpretar. generalmente se utilizan otras tecnologías para describir la presentación de una página web como CSS o la funcionalidad y comportamiento con JavaScript

CSS

CSS

es el lenguaje que usamos para diseñar un documento HTML, CSS describe cómo se deben ser renderizados los elementos de HTML en la pantalla, CSS ahorra mucho trabajo, puede controlar el diseño de varias páginas web a la vez. Su funcion es definir estilos para sus páginas web, incluido el diseño, el diseño y las variaciones en la visualización para diferentes dispositivos y tamaños de pantalla. CSS es utilizado por Bootstrap, que es un framework front-end utilizado para desarrollar aplicaciones web y sitios mobile first, o sea, con un layout que se adapta a la pantalla del dispositivo utilizado por el usuario.
